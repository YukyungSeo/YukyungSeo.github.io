- inMemory 코드를 최대한 재사용할 수 있도록 구현하였습니다. import만 바꿔주면 바로 inFile을 사용할 수 있도록 구현하였습니다.
- java 과목에서 배웠던 여러 기능들을 직접 사용해보면서 익힐 수 있었던 시간이었습니다.
